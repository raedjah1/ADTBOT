# SmartWebBot Desktop App v2.0
## Easy Installation Package

### What is SmartWebBot?

SmartWebBot is an AI-powered web automation tool that helps you:
- Automate repetitive web tasks
- Extract data from websites
- Fill forms automatically
- Test web security
- Monitor web applications

### Installation Requirements

Before installing SmartWebBot, you need:

1. **Python 3.8 or higher**
   - Download from: https://www.python.org/downloads/
   - During installation, check "Add Python to PATH"

2. **Node.js LTS version**
   - Download from: https://nodejs.org/
   - Install with default settings

### Installation Steps

1. Extract this ZIP file to any location
2. Double-click `INSTALL.bat`
3. Wait for the installation to complete
4. Double-click the "SmartWebBot" shortcut on your desktop

### First Time Use

When you first launch SmartWebBot:
- The backend server will start automatically
- The frontend will open in your browser
- Go to http://localhost:3000 to use the app

### Features

- **Dashboard**: Overview of bot status and metrics
- **Task Builder**: Create automation tasks visually
- **Live Monitor**: Watch your bot work in real-time
- **AI Assistant**: Chat with AI for task guidance
- **Security Scanner**: Test websites for vulnerabilities
- **Results Viewer**: Export and analyze collected data

### Troubleshooting

**App won't start:**
- Make sure Python and Node.js are installed
- Check that no other app is using ports 3000 or 8000
- Try running as Administrator

**Browser doesn't open:**
- Manually go to http://localhost:3000
- Check that the backend is running at http://localhost:8000

**Need help?**
- Check the logs in the terminal window
- Visit the API docs at http://localhost:8000/docs

### System Requirements

- **OS**: Windows 10 or higher
- **RAM**: 4GB minimum (8GB recommended)
- **Disk Space**: 2GB for installation
- **Internet**: Required for AI features

### Uninstallation

To remove SmartWebBot:
1. Delete the SmartWebBot folder from your Desktop
2. Delete the SmartWebBot shortcut from your Desktop
